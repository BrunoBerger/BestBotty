# js-bot
js version of BestBotty


### Install
- node 16 or higher
- create a ``` config.json```
```json
{
	"token": "your-token-goes-here",
	"guildId": "Get_ID_of_server",
	"clientId": "Get_ID_from_bot_in_server",
	"botChannelId": "Select_a_default_channel"
}
```
